Lincon

True dark HexChat theme

Get Lincon See on GitHub

Lincon is a dark HexChat theme. Customizations were made inspired by default linux console colors and irssi. I was constantly updating it over the years and decided to share it.

[Features]

  • Clean and seamless
  • Custom color scheme
  • Rewritten all text events
  • Updated to HexChat v2.10.2

[Installation]

Basic installation instructions below. For more information see HexChat appearance documentation.

[Windows]

  • Close HexChat
  • Download lincon-1.0.hct
  • Use the HexChat Theme Manager to install it.
  • Open HexChat

[Alternative Windows install]

  • Rename lincon-1.0.hct to lincon-1.0.zip
  • Extract the files to %APPDATA%\HexChat

[Linux]

  • Close HexChat
wget -c https://github.com/mc0de/lincon-hexchat-theme/releases/download/1.0/lincon-1.0.hct
unzip lincon-1.0.hct -d ~/.config/hexchat
  • Open HexChat
